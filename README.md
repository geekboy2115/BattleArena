# BattleArena

A web-based Battle Royale game built with Phaser 3.

## How to Play
- **Move**: WASD
- **Aim/Shoot**: Mouse
- **Objective**: Survive and be the last one standing!

## Features
- **Battle Royale Combat**: Fight against up to 100 CPU opponents.
- **Weapon System**:
  - **Pistol**: Unlimited ammo, basic damage.
  - **Machine Gun**: High fire rate.
  - **Shotgun**: Spread shot, high damage.
- **Looting**: Scavenge for weapons, health packs, and ammo.
- **Safe Zone**: Avoid the shrinking play area to survive.
- **High Scores**: Your max kills are saved locally.

## Development
- Uses **Phaser 3** for robust 2D rendering and arcade physics.
- Modular code structure in `script.js`.
